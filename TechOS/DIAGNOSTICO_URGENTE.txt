ESCENARIO A: PANTALLA BLANCA + ERROR EN CONSOLA
------------------------------------------------
→ Hay un error de JavaScript
→ SOLUCIÓN: Copia el error completo y repórtalo


ESCENARIO B: ERROR 404
----------------------
→ La ruta no está registrada
→ SOLUCIÓN: Verificar App.tsx (ya está bien)
→ Prueba con Ctrl+C y reinicia: npm run dev


ESCENARIO C: PÁGINA CARGA PERO DICE "NO HAY OFERTAS"
-----------------------------------------------------
→ La página funciona correctamente
→ Falta ejecutar SQL en Supabase
→ SOLUCIÓN: Ve a paso 4 abajo


ESCENARIO D: LOADING INFINITO
------------------------------
→ Supabase no responde
→ SOLUCIÓN: Verificar .env y conexión


ESCENARIO E: BOTÓN NO APARECE EN MAPA
--------------------------------------
→ Caché del navegador
→ SOLUCIÓN: Ctrl+Shift+R en /map
